# OpenTI Code of Conduct

## Our Pledge

We as members, contributors, and leaders pledge to make participation in OpenTI a harassment-free experience for everyone, regardless of age, body size, disability, ethnicity, gender identity, level of experience, nationality, personal appearance, race, religion, or sexual identity.

## Expected Behavior

- Be professional and considerate  
- Use welcoming and inclusive language  
- Respect differing opinions and experiences  
- Gracefully accept constructive criticism

## Unacceptable Behavior

- Harassment or intimidation  
- Discrimination or offensive language  
- Disrespecting contributions from others

## Reporting

If you experience or witness unacceptable behavior, please contact the project maintainers immediately via GitHub or email.

## Enforcement

Project maintainers may issue warnings or remove contributors for behavior that violates this code of conduct.
