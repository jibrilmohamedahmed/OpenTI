# OpenTI Reference Architecture

OpenTI provides the operating system layer for multi-agent AI, combining persistent cognition, deterministic orchestration, and governance.

---

## Layers

### 1. Models & Compute
- LLMs: OpenAI, Anthropic, custom models  
- Cloud or on-prem GPU clusters  

### 2. Agent Frameworks
- LangGraph  
- AutoGen  
- Enterprise internal agents  

### 3. OpenTI Cognitive Runtime
- Cognitive Memory Graph  
- Deterministic Orchestration Engine  
- Policy & Governance Engine  
- Replay & Audit Logs  
- Security & Compliance Layer  

### 4. Applications
- Capital Markets dashboards  
- Enterprise copilots  
- Government & regulatory platforms  

---

## Institutional Value

- Persistent cognition for compliance and audit  
- Replayable decision logs  
- Deterministic orchestration for operational safety  
- Standardized CMG for interoperability across institutions
